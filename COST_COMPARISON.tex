\section{Trade-offs}

\subsection{Advantages of DeepSeek-V3}
\begin{itemize}
    \item \textbf{53\% cost reduction} across all operations
    \item Comparable performance (80.6\% vs 82.0\% MMLU)
    \item Same 128K context window
    \item Open weights available for self-hosting
\end{itemize}

\subsection{Advantages of GPT-4o-mini}
\begin{itemize}
    \item Slightly better performance (1.4\% higher MMLU)
    \item More established with better documentation
    \item Potentially better instruction following
    \item Proven reliability for production use
\end{itemize}

\section{Recommendation}

For a student project with budget constraints:
\begin{itemize}
    \item \textbf{Switch to DeepSeek-V3} for RAG operations (query expansion, classification, compression)
    \item \textbf{Keep GPT-4o-mini} for transcription correction (requires high accuracy)
    \item \textbf{Hybrid approach} saves approximately \textbf{45\% monthly} while maintaining quality
\end{itemize}

\textbf{Hybrid Monthly Cost: \$11.85} (42\% savings vs GPT-4o-mini only)
